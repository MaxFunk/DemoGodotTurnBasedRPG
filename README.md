# Turn-Based RPG in Godot (Experimental)

This repository contains an **experimental RPG prototype built with Godot 4.5**, featuring a **turn-based combat system** and basic exploration mechanics.

The project is an **unfinished demo**, but it already includes a **fully functional combat system**, several enemies, and a small playable area for testing gameplay mechanics.

## Features

- **Turn-based combat system**
- Basic **exploration gameplay**
- A **small dungeon** with:
  - Multiple enemy types
  - Simple environmental puzzles
  - A boss encounter

The project serves primarily as a **technical and gameplay prototype** exploring RPG combat mechanics in Godot.

## Assets

- **3D Models and Icons**: Created by the developer of this project
- **Music**: Not owned by the developer and used only for demonstration purposes

## Running the Game

### Requirements

- **Godot 4.5 or higher**

### Controls

Currently, the game **only supports game controllers** (e.g. Xbox or PlayStation).

Keyboard input is **not implemented yet**.

### Starting the Demo

When starting the game, select a new save file. Then the player is put into a scene, that is WIP.

To access the combat testing area:

1. Walk to the **brown cube** in the starting scene.
2. **Interact with it**.
3. This will transport the player to the **Debug World**, where the combat system can be tested.
