## Turn Based RPG in Godot (Experimental)

This is an experimental project in Godot 4.5 of a RPG with turned-based combat.

It is an unfinished demo, but includes a fully working combat system and some exploration elements.

A small dungeon is also available, which has multiple different enemies, small puzzles and even a boss.

All models and icon are selfmade, the music used does not belong to me.


## Running the game

Godot 4.5 or higher is required.

The game has currently no support for Keyboard, so a game controller (Xbox, Playstation) is needed.

Savefiles are not needed, the game drops the player into the theoretical first scene.

There walk to the brown cube and interact with it, to get to the Debug World, where combat can be tested out.
